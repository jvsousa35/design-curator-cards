# AGENTS.md
## Design Prompt Engineer – Curator Card Build Instructions

You are acting as a design historian and typographic expert.

Your task is to build a single-page site (in docs/index.html) containing exactly THREE curator cards:

1. Bauhaus (1919–1933)
2. Swiss Style / International Typographic Style (1950s–1970s)
3. New Wave (Reaction to Swiss)

You must follow ALL constraints below strictly.

---

## Card Requirements (Non-Negotiable)

Each card must include:

1. A title with movement name and date range.
2. One body paragraph between 80–120 words. (The word count applies ONLY to the body paragraph.)
3. Exactly 5 vocabulary tags. Tags must be real domain terminology from design history.
4. Two real example links from credible sources (museum, archive, or documented collection).
5. One sentence beginning with:
   "Not to be confused with..."
   This must accurately clarify a common misconception.
6. One accountability sentence stating:
   "Verified using [source name]."

---

## Content Standards

- Information must be historically accurate.
- No vague language.
- No filler phrases.
- Vocabulary must match the movement correctly.
- New Wave must clearly explain how it reacts against Swiss Style.

---
